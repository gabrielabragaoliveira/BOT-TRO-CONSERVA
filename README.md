# 🛣️ Gerador Automático de Relatório TRO (Conserva)

Aplicativo em Streamlit para cruzar KMs de rodovias com fotos de execução de obras e gerar relatórios em Excel automaticamente.

## ⚙️ Como usar no Windows (Uso Local)
1. Instale o [Tesseract OCR para Windows](https://github.com/UB-Mannheim/tesseract/wiki) na pasta padrão (`C:\Program Files\Tesseract-OCR`).
2. Dê um duplo clique no arquivo `iniciar_bot.bat`. O sistema vai abrir no seu navegador.

## ☁️ Como hospedar na nuvem (Streamlit Community Cloud)
Basta conectar este repositório no Streamlit Cloud. O arquivo `packages.txt` garantirá que o Tesseract seja instalado automaticamente no servidor Linux deles.
